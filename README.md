# GYM MATCH - PO管理者専用Webアプリ

**パートナーオーナー向けジム会員管理システム**

---

## 🎯 概要

GYM MATCH PO管理ページは、パートナージムオーナー（PO）向けの専用管理システムです。会員管理、セッション管理、KPI分析などの機能を提供します。

---

## 🚀 技術スタック

- **Next.js**: 15.1.6
- **React**: 19.0.0
- **TypeScript**: 5.7.3
- **Tailwind CSS**: 3.4.17
- **Firebase**: 11.1.0

---

## ✨ 主な機能

### ✅ 実装済み

- **POログイン画面**: アクセスコード認証
- **ダッシュボード**: KPIカード表示（総会員数、アクティブ会員、休眠会員）

### 🚧 開発予定

- **会員管理**: 会員リスト、詳細情報、CRUD操作
- **セッション管理**: 予約カレンダー、セッション履歴
- **分析機能**: KPIグラフ、統計データ、CSV出力

---

## 🔧 開発環境構築

### 前提条件

- Node.js 20.x以上
- npm 10.x以上

### インストール

```bash
# 依存関係のインストール
npm install

# 開発サーバー起動
npm run dev
```

アプリは **http://localhost:3000** で起動します。
